// 文件分工
每个文件只做一件事
高内聚、低耦合——改 retriever 不影响 indexer
⚙️
server.py
MCP 入口 · 胶水层
  • 初始化 ChromaDB PersistentClient
  • 启动时调用 _auto_index() 扫描 documents/
  • 用 @mcp.tool() 暴露 3 个工具给 Claude
  • 持有全局 _collection 实例，传给 retriever / indexer
  • sys.path 注入确保模块导入无论从哪里启动
✂️
indexer.py
写入端 · 文档 → 向量
  • _read_pdf / _read_txt 读取原始文本
  • _chunk() 按 500 字 / 50 字重叠切块
  • SentenceTransformerEmbeddingFunction 向量化（paraphrase-multilingual）
  • collection.upsert() 安全写入（重复添加不报错）
  • get_or_create_collection() 设置 cosine 距离
🔍
retriever.py
读取端 · 向量 → 答案
  • search() 调用 collection.query_texts 语义检索
  • top_k 钳制在 [1, 5] 防止过量返回
  • cosine distance → similarity = 1 - distance
  • list_documents() 聚合 metadata 统计切块数和时间
  • 格式化输出供 Claude 直接整合进回答
